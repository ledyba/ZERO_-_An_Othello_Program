♪概要
　コンピュータと対戦できる，単なるオセロプログラムです．
　コンピュータ同士の対戦を見ることも出来ます．

♪起動までの準備について
　http://jp.sun.com/
　SunMicrosystemsのサイトから各自最新のJava 2 Platform
　Standard Editionをダウンロードしてインストールしてインスト
　ールしてください．その後OSによって起動方法が違います．
　　-Windows
　　同じフォルダにある「zero.exe」をダブルクリックして
　　起動してください．
　　
　　「zero-debug.exe」はデバッグ用です．
　　いろいろとメッセージが表示されます．
　　こっちのほうがオススメ．
　　-その他のOS
　　シェルで「zero.jar」のあるフォルダに移動して
　　java -jar zero.jar
　　とすれば起動できます（多分）．

♪強さ
　私よりとても強いので分かりません．
　Yahooのレーディングは1500程度まで行ったことがあります．

♪思考ルーチンのアルゴリズム
　NegaMaxにαβカットを入れてるだけです．

♪特殊な機能
　ある特定のシチュエーションを使って勝負したいときは，[Game]
   - [Edit boaad]を使うとボードの編集が出来ます．ただし，試
  合を開始した後のみです．

  ファイルに棋譜を保存します．
  同じフォルダにrecord.txtを作って書き込みます．
  自動では消しませんのでたくさんやってると凄いことになりますよ^^
  フォーマットはファイルの一番上に書いてあります．

♪その他
　"ZERO"はフリーソフトウェアです．自由に使用できます．
　実行ファイルを書き換えたりしない限り，自由に転載，二次配布
　ができます．転載時には事後でかまいませんのでメールで連絡を
　いただけると嬉しいです．
　またバグ報告，感想などをメールでいただけると嬉しいです．
